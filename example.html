Navbar Component Example

This is a working example of the navbar component. Try these features:

Features to Try

  • Hover over the logo - Watch it spin continuously
  • Hover over nav links - See the smooth highlight effect
  • Hover over social icons - See tooltips and animations
  • Try on mobile - Responsive design adapts
  • Glass effect - Notice the backdrop blur on sidebar

Quick Start

1. Copy the files to your Razor Pages project

2. Include the partial in your layout: <partial name="_Sidebar" />

3. Add the CSS and JS references

4. Customize the colors and links to match your brand
